# 🌊 GovLens

AI-powered Flood Relief Verification System

## 📌 Overview
GovLens is a system that helps verify flood relief applications using AI.

## 🎯 Problem
Manual verification is slow and can lead to fraud.

## 💡 Solution
This system uses AI to analyze documents and give decisions.

## ⚙️ Features
- Citizen login
- Upload documents
- AI decision
- Admin dashboard
